# Product Requirements Document (PRD)

# Project Name

College Compass – College Discovery & Comparison Platform

## 1. Overview

College Compass is a web platform that helps students discover, compare, and save colleges based on important factors such as fees, ratings, placements, and location.

The platform aims to simplify the college selection process by providing structured information and comparison tools in one place.

---

# 2. Problem Statement

Students often visit multiple websites to research colleges, compare fees, placements, and reviews.

This process is time-consuming and confusing.

College Compass provides a centralized platform where students can search, compare, and save colleges easily.

---

# 3. Target Users

### Primary Users

* Students seeking college admissions
* Students comparing colleges

### Secondary Users

* Parents
* Career counselors

---

# 4. Goals

### Business Goals

# 5. MVP Features

## Feature 1: Authentication

Users can:

* Register account
* Login
* Logout

### Acceptance Criteria

* User credentials stored securely
* JWT/session-based authentication
* Protected routes for saved colleges

---

## Feature 2: College Listing

Display list of colleges.

Each card contains:

* College Name
* Location
* Rating
* Fees
* Placement Percentage

### Acceptance Criteria

* Pagination support
* Responsive layout
* Data fetched from API

---

## Feature 3: Search & Filters

Users can filter colleges by:

* Location
* Rating
* Fee Range

### Acceptance Criteria

* Real-time filtering
* URL query support
* Backend filtering

---

## Feature 4: College Details Page

Display:

* Overview
* Courses Offered
* Fees
* Placement Statistics
* Ratings

### Acceptance Criteria

* Dynamic route
* Data loaded from database

---

## Feature 5: College Comparison

Users can compare 2–3 colleges.

Comparison includes:

* Fees
* Rating
* Location
* Placement Percentage

### Acceptance Criteria

* Side-by-side comparison table
* Easy add/remove colleges

---

## Feature 6: Saved Colleges

Authenticated users can:

* Save colleges
* Remove saved colleges
* View saved list

### Acceptance Criteria

* User-specific saved colleges
* Protected API routes

---
